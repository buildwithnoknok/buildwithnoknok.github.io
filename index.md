# 👋Welcome to the noknok Ecosystem

Modular hardware for Makers, 3D Designers, Builders, and Explorers.  
This is the central hub for all noknok modules, kits, documentation, and ecosystem standards.

For high-level information, please check our main site here:
- **Main Website** — [https://www.noknok.app](https://www.noknok.app)

# Who are you?

Choose your path:
- **[I'm a software maker](#software-maker)**, using existing modules and programming in Python or C.
- **[I'm a hardware maker](#hardware-maker)**, designing or modifying noknok PCBs myself.
- **[I'm a 3D designer](#3d-designer)**, creating my own housings for noknok kits.
- **[I'm a Builder](#builder)**, having fun assembling ready-made noknok kits.

---

## Software Maker
You want to **use noknok modules** to build a product, prototype, or kit and programming those in Python or C?

Start here:

- **to program using existing modules and check Example code**  
  Check the API documentation inside each module repository under `/docs/`.

- **to check and modify firmware of existing modules**  
  Available inside each module repository under `/firmware/`.

- **check the list of existing noknok modules**  
  [Module Index](#module-index)

---

## Hardware Maker  
You want to **create your own or customize (e.g. the layout of) existing hardware modules** that plug into the noknok ecosystem?

Start here:

- **Ecosystem Design Guidelines**  
  Connector standards, power rules, mechanical specs, firmware conventions.  
  [https://github.com/buildwithnoknok/ecosystem](https://github.com/buildwithnoknok/ecosystem) 

- **Module Templates**  
  Use existing modules as reference designs. Check the KiCAD design files from similar modules in the [Module Index](#module-index) below.

- **Contribution Guide**  
  How to publish your own module into the ecosystem and earn money with it.  
  [Contact the noknok Team](https://noknok.odoo.com/support#Contact-us)

---

## 3D Designer  
You want to **customize or create housings** for existing noknok modules or kits?

Start here:

- **3D Housing Files**  
  Available inside each module repository under `/hardware/3d`.

- **Primary Kit Enclosures**  
  Provided inside each kit repository under `/hardware/3d`.

- **Mechanical Standards and customization guidelines**  
  Mounting hole positions, PCB dimensions, and tolerances and recommendations regarding wall thickness, clearances, tolerances, etc.:  
  [https://github.com/buildwithnoknok/ecosystem](https://github.com/buildwithnoknok/ecosystem) 

- **check the list of existing noknok modules to modify existing designs**  
  [Module Index](#module-index)
